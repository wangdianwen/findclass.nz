# Feature: Course Detail Page - 课程详情页

> **功能模块**: 搜索与发现 | **优先级**: P0 | **排期**: Phase 1 | **状态**: 待开发

---

## 一、功能概述

### 1.1 功能描述

课程详情页是用户了解课程完整信息的核心页面，聚合展示课程介绍、教师信息、价格、上课安排等详细信息，并提供脱敏后的联系方式供用户联系。

### 1.2 核心价值

- **信息完整**: 聚合展示课程所有关键信息
- **信任展示**: 信任标识和数据来源清晰可见
- **合规脱敏**: 联系方式按规则脱敏展示
- **转化引导**: 引导用户收藏、联系或预约

### 1.3 用户故事

```
作为 家长
我希望 查看课程详情，包括教师资质、上课安排、价格等
以便 判断这个课程是否适合我的孩子

作为 家长
我希望 查看其他家长的评价
以便 了解课程的实际效果

作为 家长
我希望 收藏课程或联系教师
以便 后续做决定或咨询
```

## 四、数据模型设计

### 4.1 课程详情响应数据

```typescript
interface CourseDetailResponse {
  // 基本信息
  id: string;
  title: string;
  description: string;
  price: number;           // 每节课价格
  lessonCount: number;     // 总课时数
  originalPrice?: number;  // 原价（如果有折扣）
  
  // 评分
  rating: number;
  reviewCount: number;
  
  // 信任信息
  trustLevel: 'S' | 'A' | 'B' | 'C' | 'D';
  trustBadge: string;
  dataSource: 'first_party' | 'gumtree' | 'facebook' | 'other';
  sourceWeight: number;
  publishedAt: string;
  updatedAt: string;
  
  // 分类信息
  subject: string;
  grade: string[];
  teachingMode: 'online' | 'offline' | 'hybrid';
  language: 'chinese' | 'english' | 'bilingual';
  
  // 上课安排
  schedule: {
    days: string[];       // ['周六', '周日']
    timeSlots: string[];  // ['14:00-16:00']
    duration: number;     // 课时长（分钟）
    location: string;
    address?: string;
  };
  
  // 教师信息
  teacher: {
    id: string;
    name: string;
    title?: string;       // 职称、头衔
    bio: string;
    avatar?: string;
    verified: boolean;    // 是否平台认证
    teachingYears?: number;  // 教龄
    qualifications?: string[];  // 资质证书
  };
  
  // 脱敏联系方式
  contact: {
    phone?: string;       // 脱敏后
    wechat?: string;      // 脱敏后
    email?: string;       // 脱敏后
    wechatQrcode?: string;  // 微信二维码图片URL
  };
  
  // 标签
  tags: string[];
  
  // 用户交互
  userInteraction?: {
    isFavorited: boolean;  // 是否已收藏
    isCompared: boolean;   // 是否已加入对比
  };
}
```

## 五、API 设计

### 5.1 API 列表

| 方法 | 路径 | 功能 | 说明 |
|------|------|------|------|
| GET | /api/v1/courses/:id | 获取课程详情 | 需要课程ID |
| GET | /api/v1/courses/:id/similar | 获取相似课程 | 推荐相似课程 |

| DELETE | /api/v1/courses/:id/favorite | 取消收藏 | 需要认证 |
| POST | /api/v1/courses/:id/compare | 加入对比 | 需要认证 |
| DELETE | /api/v1/courses/:id/compare | 移出对比 | 需要认证 |
