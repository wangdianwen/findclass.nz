# Feature: Personal Teacher - 个人教师功能

> **功能模块**: 用户系统 | **优先级**: P0 | **排期**: MVP | **状态**: 草稿

---

## 一、功能概述

### 1.1 功能描述

个人教师功能允许用户申请成为独立教师，自己发布课程、管理学员、接收评价。所有数据归属于教师个人，可随时导出或带走。

### 1.2 核心特点

| 特点 | 说明 |
|------|------|
| **免费使用** | 无需付费，完全免费 |
| **自主管理** | 自己管理课程、学员、评价 |
| **数据归属** | 所有数据属于教师个人 |
| **灵活提现** | 线下收款，收入全归自己 |

### 1.3 用户故事

```
作为 有一技之长的个人
我希望 在平台展示我的教学能力
以便 吸引学员报名

作为 在职教师
我希望 业余时间接一些私单
以便 增加额外收入

作为 小型工作室（1-2人）
我希望 有一个展示窗口
以便 让学员找到我们
```

## 二、权限说明

### 2.1 个人教师 vs 机构教师

| 能力 | 个人教师 | 机构教师 |
|------|:--------:|:--------:|
| **注册方式** | 自己申请 | 机构创建 |
| **管理权** | 自己管理 | 机构管理 |
| **发布课程** | ✅ 自己发布 | ❌ 机构分配 |
| **收款方式** | 线下自行 | 平台代收 |
| **评价归属** | 个人 | 机构 |
| **数据带走** | ✅ 可以 | ❌ 不可以 |
| **费用** | 免费 | 机构付费 |
| **平台服务费** | 无 | 15% |

### 2.2 个人教师功能清单

| 功能 | 说明 | 状态 |
|------|------|------|
| 发布课程 | 自己创建、上架、下架课程 | ✅ |
| 课程管理 | 编辑课程信息、调整价格 | ✅ |
| 学员管理 | 查看报名学员、联系学员 | ✅ |
| 评价管理 | 查看学员评价、回复评价 | ✅ |
| 收款 | 线下自行（银行转账/现金） | ✅ |
| 数据导出 | 导出学员、评价、收入记录 | ✅ |
| 个人页面 | 展示教师档案、课程、评价 | ✅ |

## 七、数据归属说明

### 7.1 个人教师数据权利

| 数据类型 | 归属 | 可导出 | 说明 |
|----------|------|:------:|------|
| 个人档案 | ✅ 教师 | ✅ | 姓名、简介、资质 |
| 课程数据 | ✅ 教师 | ✅ | 课程信息、定价 |
| 学员信息 | ✅ 教师 | ✅ | 需遵守隐私保护 |
| 评价记录 | ✅ 教师 | ✅ | 评价内容、时间 |
| 收入记录 | ✅ 教师 | ✅ | 预估收入、课时 |
| 机构分配 | ❌ 机构 | ❌ | 机构教师数据 |

### 7.2 隐私保护

```
数据使用规则：
│
├── 学员信息使用
│   ├── 仅用于教学沟通
│   ├── 不得外传或滥用
│   └── 遵守隐私政策
│
├── 评价数据使用
│   ├── 可用于个人宣传
│   ├── 需获得学员同意
│   └── 不得虚假宣传
│
└── 数据导出
    ├── 定期导出备份
    ├── 安全存储
    └── 离职/退出时导出带走
```

| 退出教师 | 确认退出 | 教师身份移除，数据保留 |
