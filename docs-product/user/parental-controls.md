# Feature: Children Management - 子女管理

> **功能模块**: 用户系统 | **优先级**: P0 | **排期**: Phase 2 | **状态**: 开发中

---

## 一、功能概述

### 1.1 功能描述

子女管理是家长用户在个人中心管理子女信息的模块，支持添加、编辑、删除子女信息（姓名、性别、年级），方便在为子女申请课程时快速选择。

### 1.2 核心价值

- **便捷选课**: 为子女添加信息后，申请课程时可快速选择对应的子女
- **信息管理**: 集中管理多名子女的学习信息
- **个性化推荐**: 基于子女信息提供个性化课程推荐

### 1.3 用户故事

```
作为 家长
我希望 添加我孩子的信息（姓名、性别、年级）
以便 在为孩子报名课程时更方便

作为 家长
我希望 编辑或删除我孩子的信息
以便 保持信息的准确性

作为 家长
我希望 在申请课程时选择对应的孩子
以便 为特定的孩子报名合适的课程
```

## 二、功能架构

### 2.1 功能模块

```
子女管理
│
├── 👶 子女列表
│   ├── 展示已添加的子女信息卡片
│   └── 快捷操作（编辑、删除）
│
├── ➕ 添加子女
│   ├── 姓名（必填）
│   ├── 性别（男/女/其他）
│   └── 年级（幼儿园-高中）
│
├── ✏️ 编辑子女
│   └── 修改子女信息
│
└── 🗑️ 删除子女
    └── 确认删除
```

### 2.2 子女信息字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 姓名 | 文本(2-20字符) | 是 | 真实姓名或昵称 |
| 性别 | 选择 | 是 | 男/女/其他 |
| 年级 | 选择 | 是 | 幼儿园、小学1-6年级、初中1-3年级、高中1-3年级 |

### 2.3 年级选项

| 分类 | 选项 |
|------|------|
| 学龄前 | 幼儿园 |
| 小学 | 一年级、二年级、三年级、四年级、五年级、六年级 |
| 初中 | 七年级、八年级、九年级 |
| 高中 | 十年级、十一年级、十二年级、十三年级 |

## 三、数据模型设计

### 3.1 子女信息表

```sql
CREATE TABLE children (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),

    -- 关联
    parent_id       UUID NOT NULL REFERENCES users(id),

    -- 子女信息
    name            VARCHAR(50) NOT NULL,
    gender          VARCHAR(20),  -- male, female, other
    grade           VARCHAR(50) NOT NULL,

    -- 状态
    status          VARCHAR(20) DEFAULT 'active',

    -- 时间戳
    created_at      TIMESTAMP DEFAULT NOW(),
    updated_at      TIMESTAMP DEFAULT NOW()
);

CREATE INDEX idx_children_parent ON children(parent_id);
CREATE INDEX idx_children_status ON children(status);
```

## 四、API 设计

### 4.1 API 列表

| 方法 | 路径 | 功能 | 说明 |
|------|------|------|------|
| GET | /api/v1/children | 获取子女列表 | 返回家长所有子女 |
| POST | /api/v1/children | 添加子女 | 创建子女信息 |
| PUT | /api/v1/children/:id | 更新子女信息 | 修改子女信息 |
| DELETE | /api/v1/children/:id | 删除子女 | 删除子女信息 |

### 4.2 API 详细设计

#### 4.2.1 GET /api/v1/children

**响应示例** (200):

```json
{
  "success": true,
  "data": [
    {
      "id": "child-001",
      "name": "张小明",
      "gender": "male",
      "grade": "初中一年级",
      "createdAt": "2026-01-15T10:00:00Z"
    },
    {
      "id": "child-002",
      "name": "张小红",
      "gender": "female",
      "grade": "小学三年级",
      "createdAt": "2026-01-16T10:00:00Z"
    }
  ]
}
```

#### 4.2.2 POST /api/v1/children

**请求示例**:

```json
{
  "name": "张小明",
  "gender": "male",
  "grade": "初中一年级"
}
```
