# Feature: Institution Management Backend - 机构管理后台

> **功能模块**: 机构管理 | **优先级**: P0 | **排期**: Phase 2 | **状态**: 草稿

---

## 一、功能概述

### 1.1 功能描述

机构管理后台是为付费机构提供的运营管理平台，包含教师管理、课程管理、财务管理、数据统计等功能模块。

### 1.2 核心价值

- **统一管理**：机构运营一站式管理
- **数据归属**：教师、课程、评价数据归属机构
- **财务透明**：收入、结算清晰可查
- **决策支持**：数据统计辅助运营决策

### 1.3 用户故事

```
作为 机构管理员
我希望 管理我的教师团队
以便 合理分配教学资源

作为 机构管理员
我希望 管理我的课程
以便 统一展示机构品牌

作为 机构管理员
我希望 查看收入统计
以便 了解经营状况

作为 机构管理员
我希望 管理学员预约
以便 妥善安排上课时间
```

### 4.4 教师状态管理

| 状态 | 说明 | 操作 |
|------|------|------|
| **待激活** | 已添加，未登录过 | 重新发送邀请、删除 |
| **在职** | 正常工作中 | 编辑、分配课程、停用 |
| **停用** | 暂停服务 | 重新启用、删除 |
| **已删除** | 彻底删除 | 不可恢复 |

## 十、测试用例

| 测试场景 | 操作步骤 | 预期结果 |
|----------|----------|----------|
| 添加教师 | 输入信息 → 发送邀请 | 教师收到邮件，可激活 |
