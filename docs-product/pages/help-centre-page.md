## 一、功能概述

### 1.1 功能描述

帮助中心为用户提供常见问题解答和快速链接，帮助用户了解和使用平台。

### 1.2 用户故事

```
作为 用户
我希望 查看常见问题的答案
以便 解决使用平台时遇到的疑问

作为 用户
我希望 快速跳转到相关页面
以便 快速完成我想操作的功能
```

## 三、FAQ 数据结构

### 3.1 FAQ 项目结构

```typescript
interface FAQItem {
  key: string;      // 唯一标识
  question: string; // 问题
  answer: string;   // 答案
}
```

### 3.2 现有 FAQ 列表

| Key | 问题 | 答案 |
|-----|------|------|
| q1 | 如何搜索课程？ | 使用首页搜索栏，通过关键词、科目或地区搜索 |
| q2 | 如何联系导师？ | 点击课程详情页的联系导师按钮 |
| q3 | 如何预约试听课？ | 直接联系导师协商 |
| q4 | FindClass.nz 的付款流程是怎样的？ | 平台免费，付款直接与导师协商 |
| q5 | 如果需要取消或重新安排课程怎么办？ | 直接与导师沟通 |
| q6 | 如何举报错误的课程信息？ | 使用举报按钮或联系我们 |
| q7 | 如何成为 FindClass.nz 的导师？ | 点击加入导师按钮申请 |
| q8 | 我的个人信息如何受到保护？ | 信息加密存储，不分享给第三方 |
