# findclass.nz Backend

Course search platform backend for New Zealand. Node.js/Express API with AWS Lambda support.

## Quick Start

```bash
npm install
docker-compose up -d dynamodb-local
npm run dev
```

## Tech Stack

- Node.js 18+ / TypeScript 5
- Express.js + AWS Lambda
- DynamoDB (single-table design)
- JWT authentication
